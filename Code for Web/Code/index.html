คณะวิทยาศาสตร์และเทคโนโลยี

สาขาวิชามัลติมีเดีย หลักสูตรปริญญาตรี มุ่งเน้นการพัฒนานักศึกษาให้มีความเชี่ยวชาญด้านการสร้างสรรค์ และออกแบบสื่อมัลติมีเดียในยุคดิจิทัลด้วยการเรียนรู้เทคโนโลยีที่ทันสมัย เช่น การออกแบบกราฟิก การสร้างภาพเคลื่อนไหว เป็นต้น

โครงสร้างหลักสูตร

โครงสร้างหลักสูตรของสาขาวิชานี้ แบ่งเป็นหมวดวิชาที่สอดคล้องกับที่กำหนดไว้ในเกณฑ์มาตรฐานหลักสูตรของกระทรวงศึกษาธิการ
